Lab 6: Interactive Hometown Map

Custom Basemap Design & Web Mapping with Mapbox and Folium

Interactive Map

Explore the map below by zooming in and out. Click on markers to learn more about each location.

🎨 Design Inspiration

Basemap Style: For my design for my basemap my goal was to make it a simple to use as possible. I chose regular styles you would see on most interactive maps keeping land and water components tan (desert like) and blue. I wanted the map to feel sleep but detailed adding variations for colors for different types of roads, monuments, and geo locations. I also kept the overall palette simple so that my hometown pop up locations are clear and visible.

🗺️ Cartographic Challenges

Multi-Scale Design: My basemap adapts across zoom levels:

  • World/Regional (Zoom 1-10): One of the bigger challenges was making decisions on design for multiple zoom levels. At a wider view I wanted my map to feel clean and sleek with few visible labels, so I limited visible features to only display major roads or land usage.
  • Local (Zoom 12+): As the user zooms in closer to the neighborhood level, you can see the smaller streets, buildings, and labels. Creating this balance was tricky as I aimed to have as much data as possible while still having a efficient and easily maneuverable map.

Challenges Overcome:

  • The biggest technical challenge I faced was integrating my Map Box with the Folium in VS Code/Python. Getting both the URL format and making sure my style ID loaded in properly took a few rounds of troubleshooting. Another huge issue I ran into was my GitHub blocking my push due to Map Box token being hardcoded into my file. I had to use a (.env) file in order to store the token separately and then add it to (.gitignore) so it could not be tracked. I used tools like Claude Haiku 4.5 copilot to help me understand the environments and debug.

Creating a custom basemap isn't just about making a map look pretty—it's about intentionally communicating information and creating an experience that serves your data and audience.
